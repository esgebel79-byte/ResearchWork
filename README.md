# COVID-19 Forecasting MLOps Framework (PR-Patch)

Данный репозиторий содержит MLOps-платформу для гибридного нейрофизического прогнозирования временных рядов эпидемиологических данных (на примере COVID-19 в Санкт-Петербурге). В основе проекта лежит оригинальная архитектура **PR-Patch (Physics-Regularized PatchTST)**, дополненная анализом критических замедлений (Critical Slowing Down, CSD) и механизмами объяснимого ИИ (XAI).

Проект переведен из экспериментальных Jupyter-ноутбуков в промышленную модульную структуру, готовую к воспроизведению, сквозному тестированию и деплою.

## 📌 Ключевые особенности и научная новизна
1. **Мультивариантный пайплайн (Multivariate Pipeline):** Поддержка гетерогенных признаков — потоковых (суточный объем ПЦР-тестов), кумулятивных (выявленные случаи `CONFIRMED.sk`) и емкостных (занятые койки `OCCUPIED_BEDS_CALCULATED`).
2. **Физическая регуляризация на базе SIR-динамики:** Функция потерь `PhysicsRegularizedLoss` оценивает многомерный баланс между восприимчивыми ($S$), инфицированными ($I$) и удаленными ($R$) компонентами популяции.
3. **Латентная Cross-Attention модуляция:** Внедрен механизм многоголового внимания (Multi-head Cross-Attention), связывающий эмбеддинги эпидемических рядов с индикаторами раннего предупреждения (EWS): авторегрессионным коэффициентом $AR(1)$ и скользящей дисперсией. При приближении к точкам излома тренда (бифуркациям) модель адаптивно увеличивает вес физических ограничений.
4. **H-шаговая траекторная верификация (Cumulative PCE):** Расчет метрики Physics Consistency Error (ошибки физической консистентности) выполняется интегрально по всей прогнозной траектории на горизонт $H$ вперед.
5. **Анализ чувствительности и Парето-оптимальность:** Модуль автоматического построения фронта Парето в осях `MSE` (точность) — `PCE` (физическая адекватность) для строгого численного обоснования гиперпараметра регуляризации $\lambda$.

│   ├── models/              # Архитектуры нейросетей
│   │   ├── __init__.py
│   │   ├── lstm.py          # Baseline-модель LSTM

│   └── plots/               # Графики Парето-фронта, 3D-поверхности ошибок и SHAP-диаграммы

🛠️ Инструкция по развертыванию и запуску

1. Подготовка окружения
Убедитесь, что у вас установлен Python версии 3.10+. Склонируйте репозиторий и установите необходимые библиотеки:

git clone [https://github.com/esgebel79-byte/ResearchWork.git](https://github.com/esgebel79-byte/ResearchWork.git)
cd ResearchWork
pip install -r requirements.txt

2. Запуск интеграционного теста (Проверка размерностей)
Чтобы проверить корректность работы Cross-Attention слоев, генерацию синтетических батчей и прохождение градиентов без обращения к реальным файлам, запустите:
python integration_pipeline_test.py

3. Обогащение признаков и расчет EWSПоместите исходный файл в data/raw/ и запустите пайплайн предобработки, который автоматически применит правила агрегации (потоковые величины суммируются, емкостные и кумулятивные берутся по последнему значению) и рассчитает метрики CSD:
python -c "from src.features import enrich_features; enrich_features()"

4. Анализ чувствительности по λ и построение Парето-фронтаДля запуска обучения модели на реальном датасете по всей сетке значений гиперпараметра lambda, оценки H-шаговой ошибки консистентности в предкризисных окнах и автоматической генерации графиков в artifacts/plots/, выполните:
python -c "from src.evaluation import run_lambda_sensitivity_analysis; run_lambda_sensitivity_analysis()"

5. Интерпретация модели (XAI)Для локального анализа вклада лагов и CSD-индикаторов во время перелома эпидемических режимов с помощью SHAP и LIME:
python -c "from src.explainability import run_explainers_pipeline; run_explainers_pipeline()"
Результаты (summary plots, force plots) будут сохранены в artifacts/plots/.

📊 Научные артефакты (Outputs)
После выполнения полного цикла экспериментов в папке artifacts/ формируются:
metrics/lambda_sensitivity.csv — Таблица компромисса между физикой и точностью.
plots/lambda_sens_pareto.png — График Парето-оптимальности для защиты выбора рабочей точки lambda
plots/shap_bifurcation_*.png — Визуализация переключения внимания скрытых слоев PR-Patch на физические переменные в моменты начала новых волн заболеваемости.
